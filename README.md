# Chat-Application
A Chat Application Using Python With 256 Bit AES Encryption.Now Has VideoChat Feature.

# Module Installation :

Installing Needed Module For This Program :
```bash
pip install -r requirements.txt
```
# Usage :
The Server Side must be running first.
Server side :
```bash
python3 Server.py
```
Client Side : 
```bash
python3 Client.py
```
